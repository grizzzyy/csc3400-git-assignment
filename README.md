examples:

  - substract: 5 - 2 = 3
  - Divide: 10/2 = 5
  - Divide by zero shows an error message 
  - Square root: sqrt(9) = 3

FEATURES:
1) Basic operations: add, subtract, multiply, divide
2) Advanced operations: power, square root
3) Simple command line interface
4) Basic input validation (divide by 0, negative square root)
